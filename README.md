# Anas Syed — Portfolio

Personal portfolio site. Built with React, TypeScript, Vite, and Tailwind CSS.

**Live:** [anassyed.me](https://anassyed.me)

## Tech

- React 19
- TypeScript
- Vite 7
- Tailwind CSS 4
- shadcn/ui primitives

## Local development

```bash
npm install
npm run dev
```
